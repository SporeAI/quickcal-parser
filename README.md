## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/quickcal-parser.git
   cd quickcal-parser
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Usage

1. Paste your text containing dates and times into the input area
2. The app will automatically detect dates and times
3. Review the extracted events
4. Click 'Add to Calendar' to create Google Calendar events

## Technologies Used

- React
- chrono-node (for natural language date parsing)
- date-fns (date manipulation)
- TailwindCSS (styling)
